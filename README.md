# Free FreePik 🚀

Unlock the power of Freepik! This Chrome extension seamlessly integrates with the Freepik website to provide high quality image downloads directly from the page. No more navigating away or getting stuck behind premium walls, just click, choose your resolution, and download.

## ✨ Features

- **Direct Integration**: Replaces the standard "Get Premium" button with a functional "Download" menu.
- **Multiple Resolutions**: Choose from various sizes ranging from "Small View" to "Ultra HD Quality."

## 🛠️ How to Install

Since this is a developer version, you'll need to load it manually into Google Chrome:

1.  **Download the project folder** to your computer.
2.  Open Google Chrome and navigate to `chrome://extensions/`.
3.  In the top-right corner, toggle the **Developer mode** switch to **ON**.
4.  Click the **Load unpacked** button that appears in the top-left.
5.  Select the folder containing these files (`Free FreePik`).
6.  You're all set! The extension is now active.

## 📥 How to Use

1.  Navigate to any image detail page on [Freepik](https://www.freepik.com/).
2.  Locate the blue **"Download"** button (formerly the "Get Premium" button).
3.  Click it to reveal the resolution picker menu.
4.  Choose your desired quality.
    - _Note: High resolutions (1060px and above) will include the original Freepik watermark._
5.  Your download will start instantly!
